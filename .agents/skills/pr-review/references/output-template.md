This is the structure `scripts/format-output.sh` renders. The orchestrator does NOT write this manually — it builds the JSON input, then pipes through the formatter.

## Verdict thresholds (must match Phase 5 of SKILL.md)

| Verdict | Criterion |
|---|---|
| 🚫 **NEEDS_CHANGES** | any verified Critical, or ≥3 verified Major |
| 💬 **DISCUSS** | 1–2 verified Major, or Critical with low confidence/partial verification |
| ✅ **LGTM** | only Minor/Trivial/Info, or no findings |

## Sort order within each severity section

1. category (security > bug > performance > backcompat > ux > codesmell > dx > adversarial)
2. file path (alphabetical)
3. line_start (ascending)
